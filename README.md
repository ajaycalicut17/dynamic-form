# Dynamic Form

### Requirement
PHP Version : 8.0.9

### Installation

```bash
git clone https://github.com/ajaycalicut17/dynamic-form
```

```bash
cd dynamic-form
```

```bash
composer install
```

```bash
cp .env.example .env
```

```bash
php artisan key:generate
```

```bash
php artisan migrate --seed
```

```bash
php artisan serve
```
